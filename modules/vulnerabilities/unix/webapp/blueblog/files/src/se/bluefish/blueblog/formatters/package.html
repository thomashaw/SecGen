Package for formatting the contents of Blogs.

Some ideas for expanding this package:

  • It should be possible to supply parameters to a Formatter, like e.g. a FormatterContext.
  • Perhaps a VelocityFormatter? Its context needs to be populated, however.
  • I want to be able to include special functions in a blog, such as:
    • {google:Enterprise JavaBeans} for google links
    • {imdb:The Matrix} for imdb queries
    • {susning:August Strindberg} for Susning.nu queries (Swedish special)
    • {bluewiki:Artiklar} for bluewiki links. (Must build a BlueWiki first...)
    • {amazon:ISBN1-154-132-1} for Amazon links.
